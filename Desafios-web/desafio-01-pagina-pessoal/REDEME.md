# 🚀 Challenge 01 — Personal Page

This is the first HTML challenge from the **Nova Era Tech Frontend Formation**.

The goal of this project is to practice the fundamentals of HTML by creating a personal web page using semantic elements and a clean page structure.

---

## 🎯 Objective

Build a personal profile page containing personal information, hobbies, an image, and external links while applying HTML5 best practices.

---

## 🛠 Technologies Used

- HTML5
- CSS3

---

## 📦 Features

- 👤 Personal profile
- 📷 Profile picture
- 📝 Biography section
- 🎯 Hobbies list
- 🌐 Social media links
- 🏗 Semantic HTML structure
- 🎨 Responsive CSS styling
