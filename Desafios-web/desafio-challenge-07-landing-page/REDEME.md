# 🚀 Challenge 07 — Modern Landing Page

This project was developed as part of the **Nova Era Tech Frontend Formation**.

The goal of this challenge was to build a modern and responsive landing page using **HTML5**, **CSS3**, and basic **JavaScript**, focusing on visual organization, spacing, typography, responsiveness, and professional styling.

---

## 📖 About the Project

The project is a promotional landing page for a fictional platform called **DevFlow**.

DevFlow is presented as a platform designed to help programming students organize their studies, create goals, track progress, and maintain consistency throughout their learning journey.

The landing page includes a complete modern structure with navigation, promotional content, benefits, statistics, call-to-action sections, and a responsive footer.

## 📱 Responsive Design

The landing page was designed to work on different screen sizes.

The layout adapts for:

* Desktop computers
* Laptops
* Tablets
* Smartphones
* Small mobile devices

The responsive design was created using CSS media queries.

```css
@media (max-width: 900px) {
  /* Tablet styles */
}

@media (max-width: 600px) {
  /* Mobile styles */
}

@media (max-width: 380px) {
  /* Small mobile styles */
}
```
